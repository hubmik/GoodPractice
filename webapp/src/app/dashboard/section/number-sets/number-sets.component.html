Poniżej znajduje się zobrazowanie zbiorów liczbowych

Liczby Naturalne

Liczby całkowite dodatnie za wyjątkiem zera: N = {{'{ 1, 2, 3, 4... }'}}

Liczby Całkowite

Liczby naturalne, oraz ich ujemne odpowiedniki (liczby przeciwne), łącznie z liczbą zero:
Z = {{'{ ... -3, -2, -1, 0, 1, 2, 3... }'}}

Liczby Wymierne

Wszystkie liczby całkowite, oraz te, które można przedstawić za pomocą ułamka zwykłego p⁄q:
Q = 1 1⁄2, -5 3⁄4, 5⁄1
Formalny zapis: Q = p⁄q: p, q: ∈ Z ∧ q ≠ 0

Liczby Niewymierne

Liczby, których nie można przedstawić za pomocą ułamka zwykłego. Liczby te mają rozwinięcie dziesiętne nieskończone nieokresowe:
IQ = {{'{√2, √5, π}'}}
