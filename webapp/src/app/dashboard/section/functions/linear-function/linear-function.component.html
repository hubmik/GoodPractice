Funkcja liniowa opisana jest wzorem f(x) = ax + b, gdzie a to współczynnik kierunkowy funkcji, x to argument, b jest współrzędną punktu przecięcia z osią Oy: (0, b).
Wykresem funkcji jest linia prosta.
Funkcja liniowa może zawierać monotoniczność stałą, rosnącą, oraz malejącą.
